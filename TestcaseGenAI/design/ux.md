# UX / Wireframes

Minimum viable UI:
- Header with project title and model/provider selector.
- Textarea to paste Jira requirement text.
- Options: model, temperature, include NFRs checkbox, verbosity.
- Generate button.
- Results table listing generated test cases with quick actions: View, Export JSON, Export CSV, Push to Jira.

Review flow:
1. User generates tests.
2. Tests appear in a review table; user can open a test to edit fields.
3. User exports selected tests or pushes them to Jira after review.
